## Características
- Compatible con suscripciones de Clash, V2ray/V2fly, Sing-box, Shadowsocks, Sub, Github.
  - Soporte completo para configuración de `clash`, soporte parcial para configuración de `clash.meta`.

- Un conjunto de reglas de enrutamiento aplicadas a múltiples fuentes de suscripción selecciona automáticamente nodos eficientes.
- Soporta grupos de reglas de enrutamiento personalizados y grupos de nodos.
  - Personaliza grupos de reglas de enrutamiento por defecto para usuarios novatos - listo para usar desde el primer momento.
  - Integración de geo-IP, geo-site, ACL y [otros conjuntos de reglas](https://github.com/KaringX/karing-ruleset/)

- Copia de seguridad y sincronización, sincronizando múltiples dispositivos con una sola configuración.
  - Soporta sincronización con iCloud [IOS/MacOS].
  - Soporta sincronización dentro de la red de área local.
  - Soporta WebDAV.
  - Soporta importación/exportación de archivos ZIP.

- Soporte nativo para el [núcleo de sing-box modificado](https://github.com/KaringX/sing-box).
- Introduce un modo principiante para una configuración más sencilla.
- Plan para soportar más plataformas.

## Requisitos del sistema
- Windows >= 10 (solo 64 bits)
- Android >= 8 (arm64-v8a, armeabi-v7a)
- Linux (solo 64 bits)
- IOS >= 15
- MacOS >= 12 (Intel, Apple Silicon)
- TvOS >= 17

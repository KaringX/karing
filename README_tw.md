## 特點
- 相容 Clash、V2ray/V2fly、Sing-box、Shadowsocks、Sub、Github 訂閱。
  - 完全支援 `clash` 配置，部分支援 `clash.meta` 配置。

- 一套路由規則應用於多個訂閱源，自動選擇高效節點。
- 支援自定義路由規則組和節點組。
  - 為小白用戶定制預設路由規則組 - 開箱即用。
  - 內置 geo-IP、geo-site、ACL 和[其他規則集](https://github.com/KaringX/karing-ruleset/)

- 備份和同步，一次配置多設備同步。
  - 支援 iCloud 同步 [IOS/MacOS]。
  - 支援區域網絡內同步。
  - 支援 WebDAV。
  - 支援導入/導出 ZIP 文件。

- 內置支援[魔改版 sing-box 核心](https://github.com/KaringX/sing-box)。
- 增加新手模式，配置更簡單。
- 計劃支援更多平台。

- Linux (僅限 64 位)
- IOS >= 15
- MacOS >= 12 (Intel, Apple Silicon)
- TvOS >= 17

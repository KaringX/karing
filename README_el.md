## Χαρακτηριστικά
- Συμβατό με συνδρομές Clash, V2ray/V2fly, Sing-box, Shadowsocks, Sub, Github.
  - Πλήρης υποστήριξη διαμόρφωσης `clash`, μερική υποστήριξη διαμόρφωσης `clash.meta`.

- Ένα σύνολο κανόνων δρομολόγησης που εφαρμόζονται σε πολλαπλές πηγές συνδρομής επιλέγει αυτόματα αποτελεσματικούς κόμβους.
- Υποστηρίζει προσαρμοσμένες ομάδες κανόνων δρομολόγησης και ομάδες κόμβων.
  - Προσαρμόζει τις προεπιλεγμένες ομάδες κανόνων δρομολόγησης για αρχάριους χρήστες - έτοιμο προς χρήση αμέσως μετά την εγκατάσταση.
  - Ενσωματωμένο geo-IP, geo-site, ACL και [άλλα σύνολα κανόνων](https://github.com/KaringX/karing-ruleset/).

- Δημιουργία αντιγράφων ασφαλείας και συγχρονισμός, συγχρονισμός πολλαπλών συσκευών με μία μόνο διαμόρφωση.
  - Υποστηρίζει συγχρονισμό iCloud [IOS/MacOS].
  - Υποστηρίζει συγχρονισμό εντός του τοπικού δικτύου.
  - Υποστηρίζει WebDAV.
  - Υποστηρίζει εισαγωγή/εξαγωγή αρχείων ZIP.

- Ενσωματωμένη υποστήριξη για τον [τροποποιημένο πυρήνα sing-box](https://github.com/KaringX/sing-box).
- Εισάγει μια λειτουργία για αρχάριους για απλούστερη διαμόρφωση.
- Σχέδιο υποστήριξης περισσότερων πλατφορμών.
